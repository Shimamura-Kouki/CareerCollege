- userテーブルの残高(money)を取得：PHPプログラムであればif文等でチェック。今回はSQLのみなのでチェック処理はオミット

```

  user_id int unsigned NOT NULL,
  money int unsigned ,
  PRIMARY KEY(user_id)

```

  user_card_id int unsigned NOT NULL AUTO_INCREMENT,
